# Landing Page - App Nuoc Hoa

Scaffold Vite + React + Tailwind landing page.

Getting started:

```bash
cd web_landing
pnpm install # or npm install
pnpm dev
```

Build for production:

```bash
pnpm build
pnpm preview
```

Notes:
- Tailwind v4 referenced in package.json devDependencies. If you use different package manager, adjust commands.
- This scaffold is mobile-first and responsive. Customize colors and fonts in `tailwind.config.cjs` and `src/styles/globals.css`.
